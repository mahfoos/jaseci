- **`jac build`'s whole-program type-check gate actually gates now**: the check command's diagnostic ignore filter used substring matching, so the gate's `ignore=[".jac", ...]` argument silently dropped every diagnostic (all module paths contain ".jac") and the gate could never fail. The filter now uses the same segment-aware path matching as file discovery. Also fixed: cross-module native calls with `float`/`str` signatures marshalled through an `i64(i64...)` extern guess that put floats in the wrong register class (extern declarations now come from the resolved callee signature), and `flow`/`wait` over plain scalars no longer trips the ownership sendability rule (deep-immutable scalars are sendable by value, and `wait` is a receive, not a send).
