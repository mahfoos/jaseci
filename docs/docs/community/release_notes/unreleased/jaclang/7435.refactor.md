- **Single-source compiler analyses**: every semantic analysis now has exactly one implementation in core, and backends consume stamped facts instead of re-deriving them. One shared dataflow worklist framework hosts ownership, borrow-liveness, and RC-liveness analyses; the native backend's private solver (`RcElisionProofPass`) and its emit-time ownership classifiers (`compiler/ownership.jac`) are deleted in favor of the core `RcFactsPass`. Core layout is canonical (native consumes `field_index`/`method_index`, enum ordinals/values via `EnumLayout`, and method-override binding via `resolve_method_owner`); a single `UniNode.codespace()` resolver replaces four hand-rolled context walks; the interop manifest carries resolved signatures instead of stringified placeholder types; and the object-spatial model is computed once per module through `get_osp_model`. Passes now declare the analyses they provide and require, the schedule runner enforces those declarations per module, and a silently degraded schedule is a hard error instead of a source of missing facts.
