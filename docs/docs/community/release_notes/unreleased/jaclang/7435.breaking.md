- **Breaking: native compilation is strict about analysis facts**: the required analyses (type inference, CFG, ownership dataflow, capability checks) always run on the native pathway and every error-severity diagnostic now blocks native codegen. Programs that previously compiled natively through silent fallbacks are rejected: an annotation whose type inference cannot resolve is an error instead of a silent `i64` default, undefined type names surface at analysis time as `E2018`, and emit-time lowering failures report the new `E5092` (leaving `E5090` with a single detection source, the capability check). The cached-IR format version is bumped to 15 (symbol ownership and RC-policy facts now survive the cache, and the module key includes a schema digest), so existing `.jir` caches are invalidated once.
