- **Refactor: CLI command-extension machinery collapsed into native dispatch**: The internal `extend_command` / pre-hook / post-hook / `HookContext` / `CommandPriority` / per-arg `source` layer has been removed from the CLI registry and executor, completing the plugin-system removal (#7098). Built-in features (client, scale, shadcn, byllm) now declare their flags directly on the core `build` / `start` / `create` / `add` / `remove` commands and dispatch natively through plain helper functions, instead of bolting flags and cancel-and-hijack pre-hooks onto core commands via the old plugin seam. `jaclang.cli.shadcn.plugin` was renamed to `jaclang.cli.shadcn.cli` and `extend_builtin_commands` to `register_feature_commands`. No user-facing command or flag changes.
