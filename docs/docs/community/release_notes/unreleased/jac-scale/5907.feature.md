- **Feat: Native async drivers for MongoDB (Motor) and redis.asyncio**: `MongoBackend` now overrides `aget`/`acommit` with native Motor coroutines and `RedisBackend` overrides `aget`/`aput` with native redis.asyncio, eliminating `asyncio.to_thread` overhead for L2/L3 reads under concurrent load. `ScaleTieredMemory.acommit` coordinates the async flush path. Opt-in `_optdeps/motor.jac` guard keeps Motor optional; falls back to thread-based async when Motor is unavailable.
